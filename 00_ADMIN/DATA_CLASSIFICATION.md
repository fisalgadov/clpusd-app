# Data Classification

## PUBLIC
- Generic templates and non-sensitive documentation.

## INTERNAL
- Internal engineering notes and non-sensitive project metadata.

## CONFIDENTIAL
- Internal modeling and operational context not intended for public sharing.

## RESTRICTED
- Real FX data, API outputs, model artifacts, forecasts, reports, caches, credentials, and family-office-sensitive research.
- Production model artifacts and Git LFS-tracked files require manual review before commit/update.
- Default handling: do not commit and restrict access by default.
